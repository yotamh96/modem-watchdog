<objective>
Wire the 4 event-source producers (udev, rtnetlink, asyncinotify, kmsg) into the `_production_main` TaskGroup. Each producer is `restart_on_crash`-wrapped and its task handle is appended to a `producer_tasks: list[asyncio.Task[object]]` accumulator so plan 05.6-04's `SigtermChoreography` can receive it at SIGTERM. Producers share a single `asyncio.Queue[WakeSignal]` named `event_queue` that plan 05.6-03 will drain in the real cycle loop.

Producer factories are dependency-injectable: `_production_main(args, *, inventory_factory=None, zao_factory=None, udev_factory=None, rtnetlink_factory=None, asyncinotify_factory=None, kmsg_factory=None)`. The defaults wire real pyudev / pyroute2 / asyncinotify / `/dev/kmsg`; passing fakes lets plan 05.6-05's in-process integration test pin the wiring without monkeypatching.

This plan does NOT touch the cycle body (still a stub from plan 05.6-01); does NOT wire `SigtermChoreography` or `SighupSwapper` (plan 05.6-04 does); does NOT touch the unit file (already done in 05.6-01); does NOT change Settings.
